# Python - Data Structures

This directory contains Python projects for the ALX Higher Level Programming curriculum focusing on data structures (lists, tuples, etc.).

## Files

### 0-print_list_integer.py
Prints all integers of a list, one per line.

### 1-element_at.py
Retrieves an element from a list like in C.

### 2-replace_in_list.py
Replaces an element of a list at a specific position.

### 3-print_reversed_list_integer.py
Prints all integers of a list in reverse order.

### 4-new_in_list.py
Replaces an element in a list without modifying the original.

### 5-no_c.py
Removes all characters c and C from a string.

### 6-print_matrix_integer.py
Prints a matrix of integers.

### 7-add_tuple.py
Adds 2 tuples.

### 8-multiple_returns.py
Returns a tuple with the length of a string and its first character.

### 9-max_integer.py
Finds the biggest integer of a list.

### 10-divisible_by_2.py
Finds all multiples of 2 in a list.

### 11-delete_at.py
Deletes the item at a specific position in a list.

### 12-switch.py
Switches the values of a and b.

### 13-is_palindrome.c
Checks if a singly linked list is a palindrome (C).

### 100-print_python_list_info.c
C function that prints info about Python lists (CPython).

## Requirements

- All Python scripts should use Python 3
- All Python scripts should start with `#!/usr/bin/python3`
- All Python scripts should be executable
- Code should use `str.format()` for printing integers
