# Python - More Data Structures

This directory contains Python projects for sets, dictionaries, and lambda functions.

## Files

- 0-square_matrix_simple.py: Square values in a matrix
- 1-search_replace.py: Replace elements in a list
- 2-uniq_add.py: Add unique integers
- 3-common_elements.py: Common elements in sets
- 4-only_diff_elements.py: Elements in only one set
- 5-number_keys.py: Count keys in dictionary
- 6-print_sorted_dictionary.py: Print sorted dictionary
- 7-update_dictionary.py: Update dictionary
- 8-simple_delete.py: Delete key from dictionary
- 9-multiply_by_2.py: Multiply dictionary values by 2
- 10-best_score.py: Find key with max value
- 11-multiply_list_map.py: Multiply list using map
- 12-roman_to_int.py: Convert Roman numeral to integer
- 100-weight_average.py: Weighted average
- 101-square_matrix_map.py: Square matrix using map
- 102-complex_delete.py: Delete keys by value
- 103-python.c: CPython list and bytes info
